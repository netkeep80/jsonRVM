#include <string_type.h>

Подробное описание

Рекурсивно конструирует тип string_type из символов строки

Параметры шаблона
NТребуемый размер строки
TАккумулятор символов (string_type<...>)
tailОставшиеся символы для обработки
Заметки
Особенности:
  • Максимальная длина строки: 64 символа (ограничение макроса selch)
  • Поддерживает только строковые литералы (const char[N])
  • Все вычисления выполняются на этапе компиляции
Предупреждения
Использование индексов за пределами длины строки приводит к подстановке нуль-терминатора (selch)

Пример использования:

using MyString = as_type("Hello");
constexpr auto str = to_sv("Meta");
static_assert(MyString::size == 5, "");
