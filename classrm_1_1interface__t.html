template<class implementation_t>
class rm::interface_t< implementation_t >

Базовый шаблонный класс для реализации статического полиморфизма через CRTP.

Параметры шаблона
implementation_tТип конкретной реализации, должен быть наследником interface_t

Реализует паттерн Curiously Recurring Template Pattern (CRTP), обеспечивая:

  • Механизм связывания интерфейса с конкретной реализацией
  • Безопасный доступ к методам реализации
  • Защиту от использования несвязанного интерфейса

@method link Связывает интерфейс с конкретной реализацией (protected) @method impl Возвращает ссылку на реализацию с проверкой связывания (protected)

Заметки
Наследники должны:
  1. Связать реализацию через link() в конструкторе
  2. Реализовать чисто виртуальные методы интерфейса
  3. Использовать impl() для доступа к методам реализации
Предупреждения
Попытка использования методов интерфейса без предварительного связывания с реализацией приведёт к исключению runtime_error

Пример использования:

class MyInterface : public interface_t<MyInterface> {
public:
void foo() { impl().foo_impl(); }
};
class MyImplementation : public MyInterface {
public:
MyImplementation() { link(this); }
void foo_impl() {
// реализация
}
};
